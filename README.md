# ✈️ Registro de Equipaje - Vuelos Chile

Este es un script interactivo en **Python** diseñado para automatizar y validar el proceso de registro y clasificación de equipaje para un vuelo. El programa simula el control de un aeropuerto, asegurando que los datos ingresados sean correctos antes de generar el manifiesto de carga final.

---

## 🚀 Características

* **Validación Robusta:** Controla que la cantidad de equipajes y el peso sean valores numéricos enteros y positivos.
* **Seguridad en Tickets:** Verifica que los códigos de ticket tengan un mínimo de 5 caracteres y no contengan espacios.
* **Clasificación Automática:** Clasifica el equipaje según su peso:
    * **Bodega:** Hasta 10 KG.
    * **Cabina:** Más de 10 KG.
* **Resumen Final:** Entrega un manifiesto con el total de maletas destinadas a cada sección del avión.
